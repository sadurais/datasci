Introduction

This R Markdown document explains about the Shiny App created for viewing the analysis done on US National Ocean And Atmospheric Administration NOAA Storm database. This database documents the weather/storm events happened in US during the years 1950-2011. This app uses a filtered extract of that data and answers the questions such as: Which weather events caused the most damage to Human Life/Injuries? Which weather events caused the most damage to Property and Crops in US?

Inputs and Outputs

This app lets the user select the type of damage s/he wants to see. And also select the Top-N records of that data to be displayed in tabular form. The N choices are 5, 10 and 15.

Output will be a plot of Damage (log scale) to the Event Type (y-axis) and a table display of the top-N severe events.
